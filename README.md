## 功能介绍

- 状态码
  - 0 未到场
    - 签到（权限：候场教室，管理员）
  - 1 已签到
    - 安排（权限：各面试教室，管理员）
  - 2 准备出发
    - 开始面试（权限：候场教室，管理员）
  - 3 面试中
    - 收到任何一条评论（权限：各面试教室，管理员）
  - 4 面试结束
